🔄 Setup Free Stuff System

This page will help you set up the Free Stuff / Chessed Giveaway system for JShuk classifieds.

📋 What will be installed:

  • ♻️ Free Stuff category in classifieds
  • 📦 Pickup method options (porch pickup, contact arrange, collection code)
  • ⏰ Collection deadline functionality
  • 💝 Chessed and bundle listing options
  • 📞 Contact method preferences
  • 🔐 Pickup code generation
  • 📋 Item status tracking (available, pending, claimed, expired)
  • 📝 Item request system

⚠️ Important Notes:

  • Make sure your database is backed up before running this setup
  • This will add new columns to the existing classifieds table
  • Some sample data will be inserted for testing
  • You may need to refresh your browser cache after setup
🚀 Run Database Setup 📋 View Classifieds ➕ Post New Classified

🔧 Manual Setup (if automatic fails):

If you encounter errors, follow these steps:

Step 1: Check Current Table Structure

First, run this diagnostic script to see what exists:

sql/check_table_structure.sql

Step 2: Fix Categories Table (if needed)

If the categories table has missing columns, run:

sql/fix_categories_table.sql

Step 3: Complete Setup

Then use the simple version for the rest:

sql/add_free_stuff_system_simple.sql

Instructions for Simple Setup:

  1. Open the simple SQL file in your database manager (phpMyAdmin, etc.)
  2. Run PART 1 first (creates tables)
  3. Run PART 2 (inserts categories)
  4. Run each command in PART 3 separately (adds columns one by one)
  5. Run PART 4 (creates indexes)
  6. Run PART 5 (adds sample data)
  7. Run PART 6 (adds foreign key constraint)

Note: If any command fails with "column already exists" error, just skip that command and continue.

🧪 Testing the System:

  1. Run the database setup
  2. Go to Classifieds and look for the "♻️ Free Stuff" category
  3. Try posting a free item using Submit Classified
  4. Test the filtering by clicking "Free Items Only"
  5. View a free item and test the "Request This Item" functionality
